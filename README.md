# 📚 QA Studio

Intelligent Question-Answer Generator for Academic Excellence

## 🎯 Features

- **PDF Text Extraction**: Extract text from PDF study materials
- **Question Processing**: Parse questions from TXT files
- **Smart Answer Generation**: Generate answers based on mark allocation
- **Multiple Export Formats**: PDF and HTML exports
- **Academic Theme**: Professional, clean UI designed for students

## 📋 Question Types Supported

| Marks | Answer Length | Description |
|-------|--------------|-------------|
| 1 Mark | 2 words | Brief answers |
| 2 Marks | 2 lines | Short answers |
| 4 Marks | 5 lines | Medium answers |
| 7 Marks | 8 lines | Detailed answers |
